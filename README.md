## How to run on local

- Clone this project

  ```bash
  git clone https://github.com/azaf123/cikrak-project.git
  ```

- Go to the project folder

  ```bash
  cd cikrak-project
  ```

- Install dependencies

  ```bash
  npm install
  ```

- Run the server

  ```bash
  npm start
  ```

- Open <http://localhost:3000> with your browser to see the result.
